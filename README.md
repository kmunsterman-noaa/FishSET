# Installation #
---

To install:

   install.packages("devtools")
	 library(devtools)
	 install("PATH/TO/Directory/Containing/FishSET")

---

shiny
DBI
DT
R.matlab
foreign
geosphere
grDevices
jsonlite
lubridate
rgeos
